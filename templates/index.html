Library Management

The Library Management System is a simple and efficient web application designed to manage books digitally. It allows users to add new books, edit details, view the complete collection, and delete records with ease. Built using Flask, MySQL, HTML, CSS, and JavaScript, the system provides a clean interface and smooth user experience. With features like a show/hide book list and quick editing options, it eliminates manual record-keeping and helps maintain organized, accurate book data for students, libraries, or personal use.

- Add new books with title, author, and publication year

- View all added books in a tabular format

- Edit or delete individual book records

- Toggle visibility of the books table
